## 📌 Kurzversion (für schnelle Installation)

## 1. Download
- ZIP herunterladen & entpacken

## 2. Profilbild ändern
- Eigenes Bild als `avatar.png` speichern und ersetzen

## 3. Links ändern
- Öffne `index.html` mit Editor
- Ersetze die URLs bei Discord, Instagram, GitHub
- Und speichere sie.

## 4. Testen
- Doppelklick auf `index.html`

## 5. Online stellen
- Lade die Dateien auf GitHub Pages, Netlify oder deinen Webspace
